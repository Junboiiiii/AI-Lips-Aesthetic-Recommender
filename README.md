# AI Lips Aesthetic Recommender

## Project Overview
This project uses Artificial Intelligence to analyze lip features and recommend aesthetic lip enhancement styles. 
The system extracts lip measurements from images and uses a trained machine learning model to suggest suitable lip styles.

## Features
- Lip measurement extraction from images
- Machine learning model for aesthetic recommendation
- Dataset building and model training pipeline
- API for prediction

## Project Structure
Core/               Core processing modules
data/               Dataset
Result/             Output results
api.py              API interface
train_model.py      Model training script
build_dataset.py    Dataset preparation

## Technologies Used
- Python
- OpenCV
- Scikit-learn
- Pandas
- Joblib

## Installation
Install dependencies:

pip install -r requirements.txt

## Run the Model

python train_model.py
